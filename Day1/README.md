# Login Form Design

A simple beginner-level **Login Form UI Design** created using **HTML and CSS**.

This project is part of my frontend development practice, where I am learning how to create clean and responsive user interfaces using basic web technologies.

## 📌 Project Preview

This project contains a centered login form with:

- Username input field
- Password input field
- Login button
- Dark background theme
- Rounded input boxes
- Simple modern UI styling

## 🛠️ Technologies Used

- HTML5
- CSS3
